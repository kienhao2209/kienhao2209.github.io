Tạo IAM User và cấp quyền

Tạo IAM User

  1. Truy cập IAM Management Console
  • Truy cập AWS Console tại https://console.aws.amazon.com/
  • Ở thanh tìm kiếm phía trên, gõ IAM, sau đó chọn IAM từ kết quả.

  1. Tạo IAM User mới
  • Trong bảng điều hướng bên trái, chọn Users.

  • Nhấp vào nút Create user.

  1. Đặt tên user
  • Ở phần User name, nhập: ai-invoice-scanner-user
  • Tích chọn ô: Provide user access to the AWS Management Console.
  • Chọn mục: I want to create an IAM user.

  • Phần Console password:
    • Chọn mục: Custom password.
    • Đặt mật khẩu: Admin@123
    • Bỏ tick chọn: Users must create a new password at next sign-in.
  • Nhấn Next

Gán policy cho user

  1. Gán policy cho user
  • Ở mục Permissions, chọn Attach policies directly.

  • Nhấp vào Create policy để tạo policy.
